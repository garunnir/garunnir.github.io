역삼각함수와 주욧값(principal value) 시각적 설명

a = 0.7일 때, sin θ = 0.7을 만족하는 θ가 무한히 많습니다.
→ 하나의 입력(a)에 출력(θ)이 여러 개 → 함수가 아님
sin θ 그래프
a = 0.7 수평선
무한히 많은 해 θ
θ의 범위를 [-π/2, π/2]로 제한하면, 각 a에 대해 θ가 정확히 하나만 존재합니다.
이 구간이 바로 주욧값(principal value) 구간입니다.
제한 전 (회색 = 사용 안 함)
주욧값 구간 [-π/2, π/2]
a = 0.7 수평선
arcsin(a): a를 입력하면 주욧값 구간의 θ 하나를 출력하는 함수.
정의역: [-1, 1], 치역: [-π/2, π/2]
arcsin(a) — 일대일 대응

예: arcsin(0.7) ≈ 0.775 rad (≈ 44.4°) — 딱 하나의 값만 돌려줍니다.
